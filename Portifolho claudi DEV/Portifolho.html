Sobre Mim

Sou estudante de desenvolvimento web e estou iniciando minha jornada na área de tecnologia através do curso de Desenvolvedor oferecido pela SCTEC. Durante o curso tive contato com os conceitos fundamentais da criação de aplicações web, compreendendo como funcionam as estruturas básicas de páginas e como elas interagem com o usuário.

Aprendi a construir interfaces utilizando HTML, estilizar páginas com CSS e compreender a lógica de programação aplicada ao desenvolvimento web. Também tive uma introdução aos conceitos de servidores, APIs e persistência de dados utilizando Node.js.

Este portfólio representa meus primeiros passos na área de tecnologia e será atualizado continuamente com novos projetos, estudos e aplicações desenvolvidas ao longo da minha jornada como desenvolvedora.
